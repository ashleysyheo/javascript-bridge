## 📋 기능 목록

- [x] 게임 시작 문구 출력하기 
- [x] 다리 거리 입력 받기  
  - [x] 예외 사항:
    - [x] 3~20 사이의 숫자가 아닌 경우 
    - [x] 숫자가 아닌 문자를 입력한 경우 
- [x] 다리 생성 
    - 다리 거리 만큼 위 칸과 아래 칸 중 건널 수 있는 칸은 0과 1 중 랜덤 값을 이용해서 정함
- [ ] 이동할 칸 선택 입력 받기 
  - [ ] 예외 사항:
    - [ ] 'U', 'D'가 아닌 문자를 입력한 경우
- [ ] 게임 진행사항 출력
  - [ ] 이동한 칸을 건널 수 있다면 O로 표시하고 건널 수 없다면 X로 표시
- [ ] 게임 최종 결과 출력 
  - [ ]
- [ ] 게임 재시작/종료 여부 확인 
  - [ ] 예외 사항:
    - [ ] 'R', 'Q'가 아닌 문자를 입력한 경우 
  - [ ] 총 시도 횟수 출력 
  - [ ] 재시작하면 처음에 만든 다리로 재사용
